# База данных кинотеатра

Программа для управления фильмами в кинотеатре.

## Команды сборки:

make       - собрать программу
make run   - собрать и запустить
make clean - удалить скомпилированные файлы

## Запуск:
./cinema_db

## Функции:
1. Просмотр всех фильмов
2. Поиск по жанру/рейтингу/году
3. Добавление новых фильмов
4. Удаление фильмов
5. Редактирование информации
6. Сохранение в файл# База данных кинотеатра
